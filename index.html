Based on the most advanced programming paradigms

  • Function-level

    Pure functions with no side effects. Limited, however vast, number of operators (high order functions)

  • Reactive

    Data and its views are automatically incrementally updated whenever any data they use changes

  • Event-driven

    Control flow is split into a number of small events, whose execution order is automatically built by the platform. Vast capabilities of getting what exactly was changed are also essential.

  • Object-oriented

    Support of inheritance and polymorphism, including multiple ones for both of them. Encapsulation is also encouraged, however not necessary.

  • Constraint

    Programming with declarative rules rather than imperative checks. This rules cannot be broken and often are used to automatically build proper UI.

  • Meta

    Code templating and autogeneration. Both in IDE and in run-time.
